# GORM Advanced Features

Advanced GORM patterns: transactions, hooks, custom types, raw SQL.

## Prerequisites

- Completed [25-gorm-relationships](../25-gorm-relationships/)

## What You'll Learn

- Database transactions
- Hooks (BeforeCreate, AfterUpdate, etc.)
- Custom data types
- Raw SQL queries
- Scopes
- Soft deletes

## Running

```bash
cd 26-gorm-advanced
go mod download
go run main.go
```
