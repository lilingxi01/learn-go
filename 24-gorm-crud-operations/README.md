# GORM CRUD Operations

Master Create, Read, Update, Delete operations with GORM.

## Prerequisites

- Completed [23-migrations-setup](../23-migrations-setup/)

## What You'll Learn

- Creating records
- Querying with Where conditions

- Deleting records

```bash
cd 24-gorm-crud-operations
go mod download
go run main.go
```
