# Database Basics with database/sql

Learn SQL fundamentals and Go's database/sql package.

## Prerequisites

- Completed [20-postgresql-setup](../20-postgresql-setup/)
- PostgreSQL running locally

## What You'll Learn

- database/sql package
- SQL queries (SELECT, INSERT, UPDATE, DELETE)
- Prepared statements
- Transactions
- Connection pooling
- Error handling

## Running

```bash
cd 21-database-basics
go mod download
go run connection.go
```
